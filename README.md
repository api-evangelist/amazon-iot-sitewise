# Amazon IoT SiteWise (amazon-iot-sitewise)
AWS IoT SiteWise is a managed service that makes it easy to collect, store, organize, and monitor data from industrial equipment at scale. It helps you make better data-driven decisions, quickly compute common industrial performance metrics, and create applications that analyze equipment data.

## APIs

### AWS IoT SiteWise API
The AWS IoT SiteWise API provides programmatic access to create and manage asset models, assets, gateways, portals, and dashboards for industrial equipment monitoring and data collection.
